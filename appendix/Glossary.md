Appendix B: Glossary

3NF (Third Normal Form): A database schema design approach that reduces data duplication and ensures data integrity by removing transitive dependencies.

ACID: Atomicity, Consistency, Isolation, Durability. A set of properties that guarantee database transactions are processed reliably.

Batch: A specific quantity of a product produced during a single manufacturing/harvesting run. The primary unit of traceability in this system.

Cold Chain: A temperature-controlled supply chain necessary for preserving the shelf life of perishable foods.

DML (Data Manipulation Language): SQL commands used to query, insert, update, or delete data (SELECT, INSERT, UPDATE, DELETE).

PL/pgSQL: Procedural Language/PostgreSQL Structured Query Language. A loadable procedural language for the PostgreSQL database system that allows the creation of functions, trigger procedures, and control structures.

Quarantine: A status applied to a food batch indicating it has failed a safety inspection and is isolated from further distribution.

Trigger: A database object that automatically executes a specified procedural function when a specific event (INSERT, UPDATE, DELETE) occurs on a table.
